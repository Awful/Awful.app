Special Message From Senor Lowtax

The feature you are trying to access (Private Messaging, search, etc) is only accessible to Platinum members. We're sorry, but you cannot use this feature unless you have a Platinum account.

Q: What a ripoff! I was never told about this! Why should I have to pay $10 to be able to search, when I can do it on other forums for free?

A: That's a good point! Unfortunately, running a forum with nearly 95,000 users is a little tricky. If we allow everybody to search, then the forums would crash within nanoseconds. Trust us, we've tried it before. The vB software was never created to handle up to 4,000 simultaneous users at once, and as a result, the backend database server simply can't handle the load of everybody searching.

Let me put it to you this way: each time somebody searches for something, vB has to go through a couple MILLION threads looking for those exact search terms. It can handle one person searching. It can handle one hundred people searching. It can not, unfortunately, handle a couple thousand people searching at once. If we were to let everybody search, there would be no forums left because the database server would take its own life by throwing itself onto the I-5.

So if you'd like to send Private Messages, search, attach images, and use various other bonus features, please upgrade to a Platinum account. It's only $9.95, around the price of a two-hour movie. We think you'll get more than two hours worth of entertainment here, and your payment will help us afford the seven servers powering the forums you're using right now.

Thank you!
